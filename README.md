# Student Management System

A **full-featured Student Management System** built with **Node.js**, **Express.js**, **EJS**, and **MongoDB** using **Mongoose**. This web application allows users to **manage student records** efficiently through a simple and user-friendly interface.

---

## **Features**

- Add new student records  
- View all students  
- Update student information  
- Delete student records  
- Responsive and clean UI using EJS templates  
- Data stored securely in MongoDB  

---

## **Tech Stack**

- **Backend:** Node.js, Express.js  
- **Frontend:** EJS, HTML, CSS  
- **Database:** MongoDB  
- **ORM:** Mongoose
